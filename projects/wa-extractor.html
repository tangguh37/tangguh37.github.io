WhatsApp Timestamp Extractor

PythonTesseract OCROpenPyXLPillow
🚧 Work in Progress

Problem

Photos shared in WhatsApp groups often contain visible timestamps (e.g., delivery receipts, event photos), but extracting them manually is tedious. This tool automates the process: OCR-read timestamps from images and cross-reference them with the WhatsApp chat export to identify senders.

Architecture

┌──────────────────┐ ┌─────────────────────┐ ┌──────────────────────┐ ┌──────────────────┐ │ WhatsApp │─────▶│ Parse Export │─────▶│ Tesseract OCR │─────▶│ Excel Report │ │ Chat Export │ │ Map photo → │ │ Multi-pass per │ │ Sender, Photo, │ │ _chat.txt │ │ sender name │ │ image (grayscale, │ │ Timestamp │ └──────────────────┘ └─────────────────────┘ │ high contrast, │ └──────────────────┘ │ threshold) │ └──────────────────────┘

Planned Features

  • Parse WhatsApp export zip to extract photo-to-sender mapping
  • Multi-pass OCR with different preprocessing for each image
  • Date/time pattern recognition from OCR output
  • Excel report with sender, photo filename, extracted timestamp
  • Summary statistics by person

Tech Stack

  • Python 3 — Core programming language
  • Tesseract (tesserocr) — OCR engine for text extraction from images
  • Pillow — Image preprocessing (grayscale, contrast, threshold)
  • OpenPyXL — Excel report generation
  • python-dateutil — Flexible date/time parsing
